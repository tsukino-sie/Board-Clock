# Board Clock ⏰

전자칠판에서 사용하기 위한  
항상 화면 위에 표시되는 시계 앱

---

## 기능 ⚙️

- 다른 앱 위에 항상 표시되는 시계
- 전체 화면 앱 위에서도 표시 유지
- 드래그로 화면 어디든 이동 가능 🖐️
- 화면 표시 On / Off 토글
- 시계 크기 조절 🔍
- 시계 텍스트 색상 변경 🎨
- 시계 배경 색상 변경 (Color Picker 사용)
- 배경 불투명도 조절
- 모서리 곡률 조절  
  (둥글게 / 원형 / 각지게)
- 안드로이드 부팅 시 자동 실행 (토글 가능) ⚡
- 설정 초기화 ♻️
- 앱 자동 업데이트 지원 🔄

---

## 사용 방법 📱

1. 앱 실행 ( 초기 권한 승인 필요 )
2. 화면에 표시 토글 활성화
3. 시계를 원하는 위치로 이동
4. 설정에서 디자인과 크기 조절
5. 필요 없을 때엔 토글로 비활성화

---

## 권한 안내 🔐

- 다른 앱 위에 표시 / 시계를 상시 표시하기 위한 권한
- 부팅 시 자동 실행을 위한 권한
- 자동 업데이트 확인 및 설치를 위한 권한
